/======================================\
|Cobra 11 - Nitro Multiplayer 1.2 Alpha|
\======================================/

A patcheletlen verziójú játékon lett tesztelve.
A program futtatásához legalább .Net Keretrendszer 2.0-re van szükség.

Installation
============
Másold be a fájlokat a játék mappájába.

Useage
======
Kliens (Játékos)
1. Indítsd el a c11_mp.exe-t
2. Írd be a szerver IP címét, kattints a Connect gombra
3. Válaszd ki a Single Race-t
4. Állítsd be a következõkre:
	Traffic			-> Off
	Number of opponents	-> Többi játékosok száma
	Opponents vehicle	-> Same
5. Válaszd ki ugyanazt a pályát és jármûvet, mint a többi játékos
	

Szerver
1. Indítsd el a c11_mp_serv.exe-t
2. Add meg a játékosok számát (2-6)
A 7777-es portot meg kell nyitni a szervernek.
